# Privacy Policy

**Colour Dash**
Last updated: 2026-02-02

## Information We Collect
When you create an account to use the leaderboard feature, we collect:
- A unique nickname (user ID)
- Game scores submitted to the leaderboard

Authentication is handled using Firebase Authentication. Email addresses used by the system are automatically generated and not accessed or used for communication.

## How We Use Information
- To identify players on the leaderboard
- To store and display scores
- To secure user accounts

## Data Sharing
We do not sell or share user data with third parties.

## Data Security
All data is encrypted in transit and stored securely using Firebase services.

## Children's Information
This game is suitable for all ages but is not specifically designed for children under 13.
